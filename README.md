# Social-Sentiment-Analysis
Machine learning based social sentiment analysis system to classify emotions from text data using NLP techniques.

## Features
- Sentiment classification using machine learning
- Trained on labeled text dataset
- Jupyter Notebook based implementation
- Exported results in HTML format

## Tech Stack
- Python
- Scikit-learn
- NLP (NLTK)
- Pandas, NumPy
- Jupyter Notebook
